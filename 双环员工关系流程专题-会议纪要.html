📋 双环组织人事会议纪要 · 员工关系流程专题

📅 2026年4月17日 🏢 双环 × 北森 ⏱ 约2小时15分
1 角色与分工
角色人员负责范围
职能员工关系专员（娜娜、林红）2人职能部门（~400人）的转正、调动、离职；入职由招聘岗代做
分公司人资主管+助理6个分公司×2人=12人分公司全部入转调离（~3000人）
招聘岗两人职能入职办理手续；EHR系统录入
人资经理：小芳老师1人整体员工关系统筹、流程审核、编制把控
⚠️ 关键分工：职能：招聘负责入职手续办理，员工关系负责后续转调离。分公司：人资主管+助理负责全部（入职也由人资做，非招聘）。三方公司：与分公司角色一致。
2 入职流程（详细）

2.1 线下流程 — 应聘登记表

面试通过 → 报道当天填写

面试方式填表方式
线下面试候选人现场手写填写应聘登记表
线上面试报道当天补签纸质版

应聘登记表审批签字流程（线下纸质）

职级审批节点
4级以下签到人力资源室（小芳）结束
4-5级签到人事行政中心（魏总）
6级签到本部总经理
7级以上报到集团管理总部

2.2 入职材料收集

🔴 必收材料

  • 身份证（复印件/扫描件）
  • 银行卡信息（统一中国建设银行）
  • 学历证书（大专及以上）
  • 学信网学历验证报告 + 学位验证报告
  • 体检报告（智慧园体检，每周三；自带合格报告补贴100元）
  • 特殊岗位证书：电工证、叉车证
  • 特殊岗位：职业健康体检

🟡 可选/视情况

  • 上一家离职证明（4级以上建议，非强制）
  • 四六级证书（有则当天收集）
  • 残疾人信息（管理总部特定人员操作）

2.3 EHR系统入职登记必填项

类别字段说明
基本信息姓名、证件号、性别、出生日期、籍贯标红为必填
电子证照蓝底一寸照，没有的现场换底拍摄
身份证期限固定期限/长期，到期预警
工作信息用工形式合同/派遣/外包/实习/退休返聘/其他（派遣未使用）
薪酬类别计时/计件/其他（细分目标责任制、年薪考核制等）
岗位与部门绑定，只能选该部门已配置岗位
职务/职级M/T/P/O四类，按审批结果选（如P3=三级职员）
职位信息手动填写文本（如"人事行政专员"），用于花名册显示
财务归属部门成本分摊用（如配料员财务归属挂到车间）
岗前类别岗前/在岗/其他/挂靠（入职默认"岗前"）
日期信息进入公司日期正式上班当天
进入集团日期计算工龄津贴用
试用期默认3个月，按15号节点算到月底
合同信息合同类型固定期限/无固定期限/以完成任务为期限
签订方式电子（推送e签宝）/纸质；合同编号默认工号
协议入职协议（N合一）保密+入职协议整合推送e签宝
学历信息学校、专业、学制、学历等级文本填写；区分985/211/一本/二本/三本
家庭信息关系、姓名、工作单位、政治面貌关系为选项，其余文本

2.6 入职当天线下手续

🪪 工卡办理

  • IC卡读取卡号 → 录入EHR考勤档案 → 同步门禁
  • 当天先发手写纸质卡（临时），后期贴膜正式卡
  • 门禁每天分批次同步（如7:00/8:00）
  • 宿舍门禁需单独刷卡连接

📖 员工手册学习

  • 线上在"环湾学堂"（企业微信）学习
  • 入职流程走完后才能开通权限
  • 当天告知，后期跟踪

👕 工服/工服

  • 行政仓库统一发放
  • 线上名单到一楼前台领取

📝 岗前学习期

  • 新入职默认"岗前人员"
  • 发放《入职辅导手册》纸质版
  • 每日记录工作内容，班长签字
  • 约30天后上岗考核 → 合格 → 调动（岗前→在岗）

2.7 特殊情况

类型处理方式
三方公司人员签劳务合同（纸质），不走电子签；劳动关系归属选三方公司名称
劳务转正三方人员转为正式员工 → 重新录入、重新签劳动合同、重新算工龄
薪酬确认单单独一张表，由薪酬模块（冬梅）负责，不在应聘登记表中
3 岗位体系（EHR内）

3.1 岗位配置方式

集团马老师建基准岗位库（300+）
→
各部门HR从池中勾选适用岗位
→
新员工入职只能选已勾选岗位
  • 操作路径：组织结构 → 岗位信息 → 新增 → 从基准岗位池勾选
  • 可停用（不可删除，因有关联数据）
  • 问题：辅助工岗位细分太多（按工序命名），导致岗位池膨胀
  • "职位信息"字段（手动文本填写）实际承担了岗位细分的功能

3.2 职务/职级

要素说明
职级M/T/P/O四类，按审批结果选择（如P3=三级职员）
职位信息手动填写文本（如"人事行政专员"），花名册中显示；与岗位名称高度重叠
兼职在职位信息中写"兼XX岗位"（如小芳：集团人资室主任兼本部人资室主任）

3.3 班线配置

  • 马老师手动维护（NC65组织结构中）
  • 数据来源：分公司与TPS沟通后，HR发邮件给马老师
  • 新增/撤销/调整频率低（一年两三次）
  • 入职时由填写入职登记的HR选择班线
  • 如一分公司制造一部→热前→107线→M（班线代码）
4 转正流程
维度说明
试用期默认3个月，按15号节点算到月底（如4月17日入职→7月30日到期）
合同签订规则季度末（如1月1日入职→合同到期3月31日）
适用范围全员（含职能）均需走转正流程
线下纸质转正申请表：工作内容+自我评价
线上EHR系统发起转正申请，按职级走审批
岗前学习期转正：以《入职辅导手册》上岗考核记录为依据 → 每日填写工作内容+班长签字 → 系统上岗考核评分+实操评价 → 合格后走调动流程（岗前→在岗）
💡 待讨论：蓝领工人是否可以试用期满自动转正？减轻业务部门操作负担
5 调动流程

5.1 调动类型

类型定义发起方线下表单
组织内调配（部门内）同一部门内小科室间调动调出/调入方均可部门内调动审批单
跨部门调动如营销中心→人资行政中心调入部门HR发起跨部门调动审批单
跨公司调动如职能→第一分厂调入方HR发起跨公司调动审批单
跨基地调动如本部→嘉兴需解除+新签合同跨基地调动审批单（含薪资问询）

5.2 线上流程（EHR系统）

调入部门HR发起
→
选择：平调/晋升/降级
→
填写：调入部门/岗位/职级/调配原因
→
财务归属同步调整
→
审批

5.3 线下流程

  • 先线下签调动审批单（纸质），再发起线上流程
  • 跨公司调动：扫描传签（原单位和调入单位不在同一地点）
  • 跨基地：原单位解除合同 → 调入单位新签合同
  • 四种线下表单内容相同，仅签字人不同

5.4 调动后需处理

项目说明
社保25号统一操作花名册减员/增员（跨公司/跨基地）
话费补贴线下调整（有特殊规则）
公积金每年7月统一调整（按最低标准缴纳）
合同变更跨基地需重签合同（电子签）

5.5 晋升（通过调动流程操作）

  • 选"晋升"类型，线下有单独晋升审批单
  • 职级变更：如四级→五级，职务从"人事行政岗"变为"人事行政专员"（技师）
  • 同步调整O类技师对应职级

5.6 调动时间建议

月中调动存在但尽量引导月底统一 — "你月中调动调薪那边就不好搞"。社保25号统一操作，跨公司调动当天处理好（A减B增）。
6 电子签使用情况
场景方式说明
劳动合同新签电子签系统推送e签宝，员工签字后自动盖章生效
劳动合同续签电子签同上
跨基地调动合同电子签先解除原合同，再新签
入职协议（N合一）电子签保密+入职协议整合推送
竞业协议电子签单独发送
劳务合同（三方）纸质三方公司人员不走电子签
激励协议纸质未上线系统
7 痛点与问题

⏱️ 流程时效问题

  • 入职延迟：业务忙/流动大，有时入职3天后才录EHR → 门禁开不了、考勤打不了
  • 月底通过"新员工入职签卡"补录
  • 离职延迟：OA先发起，EHR同步滞后
  • 调动延迟：线下签完才发起线上，不同步

🔄 线上线下脱节

  • 应聘登记表：线下纸质→招聘人工录入EHR（双重操作）
  • 调动：先线下签字→再线上发起
  • 转正：纸质申请表+系统流程并行
  • "线上系统目前只是办入职的一个依据"

👷 蓝领工人流程过重

  • "他们没有任何人的东西，也不可能借这些东西"——交接可简化
  • 自动转正需求
  • 入职填写信息过多（学历/学校等对蓝领不必要）

📊 花名册维护

  • 分公司有手工花名册（因流程不及时）
  • 包含状态标记：在职/待离职/旷工等
  • 理想状态：新系统到位后取消手工

👨‍🏫 师徒制

  • 新员工有指定师傅
  • 当前手工记录，系统无此功能
  • 薪资涉及师傅带徒补贴

💰 激励协议

  • 分公司一线入职签"激励协议"（补贴相关）
  • 未录入系统，纸质线下签
  • 操作工工序补贴/辅助工固定补贴
